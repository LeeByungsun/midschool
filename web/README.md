# Web workspace

이 폴더는 `MisSchoolApp`의 웹 구현을 위한 시작 위치입니다.

현재는 아직 구체적인 프레임워크를 넣기 전 단계이며, 우선 기준 문서는 아래를 따릅니다.

- `docs/project_specification.md`
- `web/AGENTS.md`
- `docs/skills/web-automation-map.md`

권장 원칙
- 공통 도메인 요구사항은 `docs/project_specification.md`의 **공통 요구사항** 섹션을 기준으로 구현
- 웹 전용 UX/기술 선택은 같은 문서의 **Web 전용 명세** 섹션을 기준으로 구체화
- Android 전용 구현 세부사항은 그대로 가져오지 말고, 웹 플랫폼 제약에 맞게 대응
