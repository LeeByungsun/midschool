# Web UI Builder

웹 대시보드/상세 화면/공통 UI를 구성할 때 사용하는 스킬입니다.

## 목표
- 모바일 퍼스트 레이아웃
- 대시보드형 정보 구조
- 재사용 가능한 카드/섹션 컴포넌트
- 웹 플랫폼에 맞는 상호작용

## 우선 대상
- 홈 대시보드
- 시간표 페이지
- 일정 페이지
- 설정 페이지
- 타이머 영역

## 원칙
- Android UI를 그대로 옮기지 않고 웹식으로 재구성
- 시맨틱 마크업과 접근성 우선
- 복잡한 디자인 시스템은 초기에 과도하게 만들지 않음
