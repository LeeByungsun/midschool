# Web PWA

웹을 설치 가능한 앱 경험으로 확장할 때 사용하는 스킬입니다.

## 목적
- PWA 적용 가능성 평가
- 설치 UX 준비
- 오프라인/캐시 전략 검토
- 앱 같은 진입 경험 강화

## 고려 항목
- manifest
- 아이콘
- 설치 배너/안내
- 오프라인 캐시 전략
- 모바일 홈 화면 진입 UX

- 초기 버전에서 무리하게 범위를 넓히지 않음
- 기본 서비스 기능이 안정된 후 적용 우선
